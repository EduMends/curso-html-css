Outras Formatações

Código-fonte

Quando existir um texto que possua um código-fonte de alguma liguagem
usa uma tag especófica para isso.

O comando document.getElementById('teste')
é escrito em linguagem JavaScript.

Usamos a tag <code> para adicionar uma fonte monoespaçada,
para assim facilitar a leitura de Códigos-Fonte em meio
a um texto de uma página.

Citações Simples

Quando ouver necessidade de citar uma frase com aspas, usamos a TAG <q>.

Exemplo: Uma vez fulano me disse Estudo programação para me profissionalizar.

Citações Completa

Para citações de textos, usamos a TAG <blockquote>.
Assim podemos citar um paragrafo de um texto de um autor.

Exemplo

Quando você entra em uma situação tensa, e tudo vem contra você até parecer que você não conseguiria aguentar nem mais um minuto sequer, não desista, pois esta é justamente a situação e momento em que a maré irá virar.

OBS: O parâmetro cite pode ser usado para adicionar o link
da fonte de origem do texto.

Abreviações

Para da signifcado a uma abreviação, usamos a taf <abbr>.

Exemplo: Estou estudando HTML5 e CSS3.
