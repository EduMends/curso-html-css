Trabalhando com Listas

Listas Ordenadas

Para listar itens em uma ordem usando marcadores, temos a tag <ol> Ordered List.
Podemos usar o parâmetro type para adicionar marcadores a lista.

Exemplo
<ol type=>

Item da lista

A tag usada para listar os itens da lista é a <il>.

Uma lista de tarefas no navegador ficaria algo asssim:

  1. acordar as 6h
  2. jogar o lixo
  3. fazer exercicios
  4. sair para o trabalho as 7h

Listas não Ordenadas

Para listas não ordenas, ou seja, sem marcadores sequênciais, usamos a tag <ul> .Unordered List.

Como nas listas ordenadas, podemos usar também o parâmetro type
com os marcadores disc,circle,square.
